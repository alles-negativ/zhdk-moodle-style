# Anpassung des Moodle Themes

## Änderungen im Admin-Backend für Boost Union

- Darstellung/Erscheinungsbild/SCSS: unter Externe SCSS Quelle auf Öffentliche URL gesetzt, und URL für externes Post-SCSS auf [File](https://raw.githubusercontent.com/alles-negativ/zhdk-moodle-style/refs/heads/main/post.scss) gesetzt, um der Seite Zugriff auf die zusätzlichen CSS-Modifikationen zu geben.
- Darstellung/Erscheinungsbild/Ressourcen: Innerhalb von Ressourcen haben wir die in diesem Repository verfügbare benutzerdefinierte Schriftart zu Nutzerdefinierte Schriftarten hinzugefügt. Diese Schriftart kann später im CSS implementiert und verwendet werden. Siehe post.scss als Referenz. Idee ist die Icons anzupassen. Standart Schrift der Webseite ist Helvetica.
- Darstellung/Erscheinungsbild/Webseite: hier haben wir die Farben wie folgt geändert:
  - Markenfarbe: #000
  - Bootstrap-Farbe für Erfolg: #91CBA2
  - Bootstrap-Farbe für Info: #05F5FA
  - Bootstrap-Farbe für Warnung: #AD6200
  - Bootstrap-Farbe für Gefahr: #FF9F9F
